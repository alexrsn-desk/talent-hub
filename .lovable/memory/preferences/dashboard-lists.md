---
name: Dashboard compact list pattern
description: All dashboard list sections must use compact expandable rows — no cards, no nested boxes
type: preference
---
Every list section on the dashboard must use the compact expandable row pattern:
- Collapsed: single tight row, 40-44px height
- Expanded: full detail in place, no new page
- No cards inside sections
- No boxes inside boxes
- Single 1px divider between rows only
- Applies to: AI Actions, Signals Detected, Actions You May Have Missed, My List, Priority Candidates, Recent Activity
- Any future dashboard feature must follow this same pattern
- Never use card components inside dashboard lists
