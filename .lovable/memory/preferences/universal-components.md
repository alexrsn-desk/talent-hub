---
name: Universal interactive components
description: All interactive elements must work identically everywhere they appear in the CRM
type: preference
---
Any interactive element — signals, action buttons, thumbs up/down, edit, expand — must work identically regardless of where it appears.

If a feature works on Calls & Meetings, it must work on candidate profiles, client profiles, job records, BD pipeline, and dashboard activity feed.

Never build a feature that only works in one place. Always build it as a reusable component that works everywhere that type of content appears.
